
README

Prior to running the scripts below, execute the following:
    
  mkdir runs && cd runs
  wget -r -np -nH -R "index.html*" --cut-dirs=2 http://mag.gmu.edu/git-data/ccmc/Chigomezyo_Ngwira_092112_3a/

Edit Chigomezyo_Ngwira_092112_3a.py, change the value of data_dir to the "runs" directory
created above.

Execute the files in the following order:
    
    1) Chigomezyo_Ngwira_092112_3a_2Dplots.py
    2) Chigomezyo_Ngwira_092112_3a_2Dnedplots.py
    3) Chigomezyo_Ngwira_092112_3a_Findboundaries.py
    4) Chigomezyo_Ngwira_092112_3a_deltaB_by_region.py
    5) Chigomezyo_Ngwira_092112_3a_heatmaps.py
    6) Chigomezyo_Ngwira_092112_3a_solar_wind.py
